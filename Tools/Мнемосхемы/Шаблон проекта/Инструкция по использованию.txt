1. Открыть проект MShema.dpr.
2. Добавить его в репозитарий (Project - Add to Repository...) с именем (Title) "Шаблон мнемосхемы"
3. Для создания новой мнемосхемы на основании шаблона выполнить: 
	File - New - Other...	
	Шаблон находится в ветке "Delphi Projects"
	Выбрать "Шаблон мнемосхемы" 
	В окне выбора каталога (Select Directory) указать каталог, в котором будет размещаться новая проектируемая мнемосхема (например, D:\Projects\Мнемосхемы\ПК №2\Водоподготовка

Примечаение: перед открытием главной формы должен быть открыт модуль uMnemoShema, в противном случае возможно появление ошибки : Error creating form: Ancestor for 'TfmMnemoShema' not found. 

4. В окне Project Manager (View - Project Manager) следует переименовать созданный проект.
5. Файл MShema.ini содержит минимально необходимые настройки. Этот файл-шаблон необходимо скопировать в каталог нового проекта и переименовать в соответствии с названием проекта (например, для проекта WaterPrep.exe этот файл должен называться WaterPrep.ini). Затем следует проверить и при необходимости изменить параметры указанные в этом файле.
